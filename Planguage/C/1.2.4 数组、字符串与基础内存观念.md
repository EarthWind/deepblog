# 数组、字符串与基础内存观念

学完变量和函数之后，下一步就要处理“批量数据”。数组让你能存一组同类型数据，字符串则是字符数组的一种特殊用法。这个阶段开始真正接触内存布局的直觉。

## 本阶段目标

- 理解数组的定义、初始化和遍历。
- 理解字符串本质上是以 `\0` 结尾的字符数组。
- 会使用常见字符串处理函数。
- 初步建立“数据在内存中连续存放”的观念。

## 核心知识

### 1. 数组是什么

数组是一组相同类型元素组成的连续内存空间。

例如：

```c
int scores[5] = {90, 85, 76, 88, 92};
```

这表示分配了 5 个连续的 `int` 空间。

### 2. 数组下标

数组从 `0` 开始编号：

- `scores[0]` 是第一个元素
- `scores[4]` 是最后一个元素

初学者最容易犯的错就是越界访问。

### 3. 字符串的本质

C 语言没有内建“字符串类型”，字符串通常就是字符数组：

```c
char name[] = "Alice";
```

它在内存里其实是：

```text
'A' 'l' 'i' 'c' 'e' '\0'
```

末尾的 `\0` 很关键，它告诉处理函数“字符串到这里结束了”。

## 可运行示例：统计数组最大值、最小值和平均值

```c
#include <stdio.h>

int main(void) {
    int scores[5] = {90, 85, 76, 88, 92};
    int i = 0;
    int max = scores[0];
    int min = scores[0];
    int sum = 0;

    for (i = 0; i < 5; i++) {
        if (scores[i] > max) {
            max = scores[i];
        }
        if (scores[i] < min) {
            min = scores[i];
        }
        sum += scores[i];
    }

    printf("max = %d\n", max);
    printf("min = %d\n", min);
    printf("average = %.2f\n", sum / 5.0);
    return 0;
}
```

## 可运行示例：字符串长度统计

```c
#include <stdio.h>
#include <string.h>

int main(void) {
    char text[100] = "deepblog";

    printf("text = %s\n", text);
    printf("length = %u\n", (unsigned)strlen(text));

    return 0;
}
```

## 示例说明

- `%s` 用于输出字符串。
- `strlen` 统计的是 `\0` 之前的字符数。
- `char text[100]` 表示数组容量是 100，但当前字符串长度不一定是 100。

## 常见误区

### 把数组长度和字符串长度混为一谈

```c
char buf[100] = "abc";
```

- 数组容量是 `100`
- 字符串长度是 `3`

### 数组越界

```c
int a[5];
a[5] = 100;
```

这是非法访问，因为合法下标只有 `0` 到 `4`。

### 忘记为字符串结尾留出 `\0`

```c
char s[3] = "abc";
```

这不够安全，因为 `"abc"` 实际需要 4 个字符位置。

## 学习建议

- 每次遍历数组时都明确边界。
- 每次处理字符串时都想一遍：末尾有没有 `\0`。
- 学字符串函数时，不要只背名字，要配小例子反复运行。

## 一个手写 `strlen` 的简单示例

```c
#include <stdio.h>

int my_strlen(const char str[]) {
    int len = 0;

    while (str[len] != '\0') {
        len++;
    }

    return len;
}

int main(void) {
    char word[] = "memory";
    printf("length = %d\n", my_strlen(word));
    return 0;
}
```

这段代码很适合帮助你理解：字符串处理的本质，就是从首地址开始逐个字符读取，直到遇到 `\0` 为止。
