# 基础语法与流程控制

这一阶段解决的是“怎样把一个想法写成能运行的程序”。你会开始接触变量、类型、输入输出、运算符和分支循环，这些内容构成了几乎所有 C 程序的基础骨架。

## 本阶段目标

- 理解变量和数据类型的意义。
- 能使用 `printf` 和 `scanf` 完成基础输入输出。
- 掌握运算符和表达式的基本规则。
- 能用分支和循环描述简单业务逻辑。

## 核心知识

### 1. 变量与数据类型

变量本质上是“有名字的存储空间”。不同类型决定了：

- 这块空间占多少字节
- 能存什么范围的值
- 参与运算时如何解释

常见基础类型：

- `int`：整数
- `char`：字符
- `float`：单精度浮点数
- `double`：双精度浮点数

### 2. 输入输出

最常用的两个函数：

- `printf`：输出
- `scanf`：输入

学习初期要重点关注格式控制符是否匹配：

- `%d` 对应 `int`
- `%c` 对应 `char`
- `%f` 常用于 `float`
- `%lf` 常用于 `double`

### 3. 运算符

需要重点掌握：

- 算术运算：`+ - * / %`
- 关系运算：`> < >= <= == !=`
- 逻辑运算：`&& || !`
- 赋值运算：`= += -= *= /=`

### 4. 流程控制

程序不是只能从上到下顺序执行，还可以：

- 用 `if` 做条件判断
- 用 `switch` 做多分支选择
- 用 `for`、`while`、`do...while` 做循环

## 可运行示例：成绩分级程序

```c
#include <stdio.h>

int main(void) {
    int score = 0;

    printf("Please input score: ");
    scanf("%d", &score);

    if (score < 0 || score > 100) {
        printf("Invalid score.\n");
    } else if (score >= 90) {
        printf("Grade: A\n");
    } else if (score >= 80) {
        printf("Grade: B\n");
    } else if (score >= 70) {
        printf("Grade: C\n");
    } else if (score >= 60) {
        printf("Grade: D\n");
    } else {
        printf("Grade: E\n");
    }

    return 0;
}
```

## 示例说明

- `scanf("%d", &score);` 中的 `&score` 表示把输入结果写入变量 `score` 的地址。
- `if...else if...else` 用于按条件区间分流。
- `||` 表示逻辑或，只要一边为真，整体就为真。

## 再看一个循环示例：1 到 100 求和

```c
#include <stdio.h>

int main(void) {
    int sum = 0;
    int i = 0;

    for (i = 1; i <= 100; i++) {
        sum += i;
    }

    printf("sum = %d\n", sum);
    return 0;
}
```

这段代码非常适合练习：

- 变量初始化
- `for` 循环结构
- 累加思想
- 输出结果

## 常见误区

### `=` 和 `==` 混淆

- `=` 是赋值
- `==` 是比较

很多初学者会在 `if` 判断里误写成赋值。

### `scanf` 不写地址符

例如把：

```c
scanf("%d", &score);
```

误写成：

```c
scanf("%d", score);
```

这会导致未定义行为，因为 `scanf` 需要一个地址来写入数据。

### 整数除法误解

```c
int a = 5 / 2;
```

结果是 `2`，不是 `2.5`。因为两个整数相除，结果仍然按整数处理。

## 学习建议

- 先把 `if`、`for` 写熟，再看更复杂的逻辑。
- 每学一个语法点都配 2 到 3 个小练习。
- 对运算结果不确定时，直接写代码验证，不要靠猜。
