# 结构体、共用体、枚举与自定义类型

当基本类型已经不够表达一个对象时，就需要自定义数据结构。比如“学生”不是一个单独的整数，而是姓名、年龄、分数等信息组合在一起。这一阶段的重点就是学会组织复杂数据。

## 本阶段目标

- 掌握 `struct` 的定义和使用。
- 理解 `enum`、`union` 和 `typedef` 的作用。
- 学会从“数据建模”的角度组织程序。

## 核心知识

### 1. 结构体 `struct`

结构体用于把多个不同类型的数据组合成一个整体。

例如学生信息：

```c
struct Student {
    char name[20];
    int age;
    double score;
};
```

### 2. 结构体成员访问

- 普通变量用 `.` 访问
- 结构体指针用 `->` 访问

### 3. `typedef`

`typedef` 用于给类型起别名，让代码更易读。

```c
typedef struct Student Student;
```

这样以后就可以直接写 `Student s;`

### 4. `enum`

枚举适合表示一组有限离散值，例如状态、等级、星期等。

### 5. `union`

共用体中的所有成员共用同一块内存，适合节省空间或做特殊底层表示，但初学阶段先理解概念即可。

## 可运行示例：学生信息结构体

```c
#include <stdio.h>

typedef struct {
    char name[20];
    int age;
    double score;
} Student;

int main(void) {
    Student s = {"Alice", 20, 95.5};

    printf("name  = %s\n", s.name);
    printf("age   = %d\n", s.age);
    printf("score = %.1f\n", s.score);

    return 0;
}
```

## 可运行示例：结构体数组

```c
#include <stdio.h>

typedef struct {
    char name[20];
    int age;
} Student;

int main(void) {
    Student students[3] = {
        {"Tom", 18},
        {"Jerry", 19},
        {"Lucy", 20}
    };
    int i = 0;

    for (i = 0; i < 3; i++) {
        printf("%s - %d\n", students[i].name, students[i].age);
    }

    return 0;
}
```

## 示例说明

- 结构体可以像普通变量一样定义和初始化。
- 结构体数组非常适合表示“多条记录”。
- 这为后续的通讯录、成绩系统、链表节点打下基础。

## 常见误区

### 把结构体当成“只是多个变量拼起来”

结构体不仅是多个变量的集合，更是对现实对象的抽象。学习阶段要开始思考“一个对象应该有哪些字段”。

### 不理解 `.` 和 `->` 的区别

- `s.age`：`s` 是结构体变量
- `p->age`：`p` 是指向结构体的指针

### `enum` 只当常量表看

枚举除了可读性更好，也有助于限制程序状态范围。

## 学习建议

- 用结构体重新建模你之前写过的小程序。
- 设计字段时优先考虑“这个对象真实需要什么数据”。
- 初学时先把结构体和数组结合起来用熟。

## 一个简单的枚举示例

```c
#include <stdio.h>

enum Level {
    LEVEL_LOW,
    LEVEL_MEDIUM,
    LEVEL_HIGH
};

int main(void) {
    enum Level level = LEVEL_HIGH;

    if (level == LEVEL_HIGH) {
        printf("current level: high\n");
    }

    return 0;
}
```

这个例子虽然简单，但体现了一个很重要的思想：程序中的很多值，并不只是数字，更应该是“有含义的状态”。
